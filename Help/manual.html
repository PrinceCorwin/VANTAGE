1. Getting Started

What is MILESTONE?

MILESTONE is a construction activity tracking and work package generation system designed for field engineers working on industrial construction projects.

Logging In

Screenshot: Login screen (start-login-screen.png)

Enter your username and password, then click Login.

The Main Interface

Screenshot: Main interface overview (start-main-interface.png)

The main window consists of:

  • Toolbar — Navigation buttons for each module
  • Content Area — The current module view
  • Status Bar — Shows current user, sync status, and version

Keyboard Shortcuts

Shortcut Action
F1 Open Help / AI Sidebar
Esc Close sidebar

5. Progress Books

Overview

Progress Books are printable checklists showing activities grouped by Area, Module, or Drawing. Field crews use these to mark completed items, which are then entered into MILESTONE by Field Engineers.

7. Reference

Glossary

Term Definition
Activity A single trackable unit of work (e.g., weld, bolt-up, cable pull)
FE Field Engineer
Lookahead A rolling schedule view showing upcoming work
P6 Primavera P6 scheduling software
ROC Rate of Completion
Sync Upload/download data to central database
Work Package A bundle of documents for a specific scope of work
